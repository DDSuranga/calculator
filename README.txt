# Online Calculator – All-in-One Tool
A fully responsive web-based calculator with multiple tools including basic math, scientific operations, unit conversion, currency exchange, age calculation, and more.

---

## 📁 Project Structure Overview

- `index.html`        → Main HTML file (this file)
- `styles.css`        → Stylesheet for layout, buttons, and themes
- `script.js`         → JavaScript logic for all calculators
- `manifest.json`     → PWA manifest for installable web app support
- `service-worker.js` → Service worker for offline caching and PWA support
- `favicon.ico`       → Website favicon
- `/css/`, `/js/`, etc. → Optional asset folders (if used)

---

## 🧰 Features Included

1. **Basic Calculator** – Standard arithmetic operations
2. **Scientific Calculator** – Trigonometric, logarithmic, and exponential functions
3. **Unit Converter** – Length, weight, temperature units
4. **Currency Converter** – Real-time exchange rate support (via API)
5. **Age Calculator** – Calculates age from birthdate
6. **Date Difference** – Days between two dates
7. **Time Difference** – Hours, minutes, seconds between timestamps
8. **Percentage Calculator**
9. **Percentage Change**
10. **Tip Calculator**
11. **VAT / Discount Calculator**
12. **Memory Functions (M+, M-, MRC)**
13. **PWA Ready** – Installable on mobile devices
14. **Responsive Design** – Mobile-friendly layout

## ⚙️ Customization Tips

To customize this calculator:

### HTML
- Add/remove calculator types inside `<main class="calculator-container">`
- Update sidebar button labels or add new ones in `.sidebar`

### CSS
- Modify colors, fonts, shadows in `styles.css`
- Adjust spacing, grid columns, padding/margin for layout tuning

### JS
- Extend functionality in `script.js`
- Add keyboard input support or integrate external APIs (e.g., live currency rates)

---

## 🧪 How to Test Locally

1. Open `index.html` in any modern browser.
2. Use browser dev tools to test responsiveness.
3. Ensure `script.js` and `styles.css` are correctly linked.
4. For full PWA functionality:
   - Serve via local server (e.g., Live Server in VS Code)
   - Test install prompt and offline behavior
